# PROYECTO-L-GICA-Katas-de-Python_Diego-Rodriguez

Este proyecto contiene una serie de 41 ejercicios de Python diseñados para cubrir conceptos fundamentales de la programación hasta funciones avanzadas. El objetivo es demostrar cómo se pueden combinar y aplicar los conceptos de programación en situaciones y funciones reales y prácticas.
Para realizar este proyecto solo es necesario instalar Python y Visual Studio.
Las primeras tareas introducen conceptos básicos de Python (variables, funciones, parámetros y manejo inicial de excepciones). Más adelante, se profundiza en validación de datos, filtrado avanzado y funciones lambda. En los últimos ejercicios ya se trabaja manipulando datos, cadenas de texto y estructuras complejas. Y se termina programando clases.
Tras la realización de este, se han comprendido los fundamentos principales de Python. Empezando desde las variables básicas como int, float o str, hasta las distintas estructuras de control (if o else). Desde mi experiencia con estas actividades, destacaría la gran utilidad de las funciones de orden superior como map(), filter() o reduce(). Del mismo modo, los métodos split(), join(), replace() o lower().
Otro aspecto fundamental que considerar durante el desarrollo de cualquier actividad de programación es la previsión y gestión de errores inesperados. Es común que durante la ejecución de un programa surjan situaciones que no funcionan como se esperaba inicialmente. Por este motivo, el manejo robusto de excepciones se convierte en un elemento clave para garantizar la estabilidad y confiabilidad del código.
